
# Zevi Store Website

Frontend for an E-commerce website


## Run Locally

Clone the project

```bash
  git clone https://github.com/akshat-9825/zevi_store.git
```

Go to the project directory

```bash
  cd zevi_store
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
